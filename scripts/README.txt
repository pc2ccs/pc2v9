The scripts in this folder are used to support interactive and sandbox problems.
pc2sandbox.sh - script to execute a submission in a sandbox
pc2syscheck.sh - check if system supports sandbox execution (exit code 0 means yes)
pc2_interactive.sh - script to execute an interactive submission with an interactive validator if not using sandboxes
pc2sandbox_interactive.sh - script to execute an interactive submission in a sandbox but the interactive validator not in sandbox
pc2validate_interactive.sh - Copy interactive results to pc2 specified location - this is the validator that is used after
	an interactive submission has completed.

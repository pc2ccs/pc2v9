About PC2 WTI

PC^2 is the Programming Contest Control System developed at California State University, Sacramento (CSUS) in support of Computer Programming Contest activities of the International Collegiate Programming Contest (ICPC) and its Local and Regional Contests around the world.

This web interface was created by Josie Isaacson, Ethan Holman, Dani Frodge, and Andrew Combs as part of their Senior Project at Eastern Washington University.
